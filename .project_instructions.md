# Siniestro - Horror Podcast App

## Overview
Premium horror podcast mobile app in Spanish with React Native Expo frontend + NestJS backend.

## Tech Stack
- **Backend**: NestJS + TypeScript + Prisma + PostgreSQL, port 3000
- **Frontend**: React Native (Expo SDK 54) + expo-router + react-native-paper
- **Auth**: JWT (email/password), tokens in SecureStore (mobile) / localStorage (web)
- **Audio**: expo-av with position saving every 15s
- **File Upload**: S3 presigned URLs

## Theme
- Dark (default): bg #0A0A0A, surface #121212, card #1A1A1A
- Light: bg #F5F0EB, surface #FFFFFF
- Primary: Blood Crimson #DC2626, Accent: Phantom Purple #7C3AED
- Gradient buttons: ['#DC2626', '#7C3AED']
- All UI text in Spanish

## Key Routes
- Auth: /api/signup, /api/auth/login, /api/auth/me
- Podcasts: /api/podcasts, /api/podcasts/:id
- Episodes: /api/episodes/recent, /api/episodes/:id
- Search: /api/search?q=&categories=
- Favorites: /api/favorites, POST/DELETE /api/podcasts/:id/favorite
- Ratings: POST /api/podcasts/:id/rating (upsert)
- Comments: GET/POST /api/episodes/:id/comments, DELETE /api/comments/:id
- Playback: POST /api/episodes/:id/playback
- Categories: /api/categories
- Upload: /api/upload/presigned, /api/upload/complete, /api/files/:id/url

## Frontend Navigation (expo-router)
- app/_layout.tsx: Root with ThemeProvider, AuthProvider, PlayerProvider, MiniPlayer
- app/auth/: login.tsx, signup.tsx (redirect to /tabs/home if authenticated)
- app/tabs/: home, search, favorites, profile (redirect to /auth/login if not authenticated)
- app/podcast/[podcastId]/: Podcast detail with rating, favorites
- app/episode/[episodeId]/: Episode detail with comments
- app/player.tsx: Full-screen audio player (modal)
- app/edit-profile.tsx: Edit user profile with avatar upload

## Seed Data
- 6 Spanish horror podcasts across 5 categories, 3-5 episodes each
- Demo user: demo@siniestro.com / password123
- Placeholder cover images via picsum.photos, placeholder audio via soundhelix
